¿Que es HTML?

HTML es es un lenguaje de marcado, lo que significa que utiliza etiquetas y elementos para definir la estructura y el contenido de una página web. Estas etiquetas se utilizan para marcar diferentes tipos de contenido, como encabezados, párrafos, enlaces, imágenes, formularios y mucho más. Cada etiqueta tiene una función específica y se utiliza para organizar y presentar información de manera coherente en un navegador web.

¿Que se puede hacer en html?

  1. Crear estructura de página
  2. Incorporar contenido multimedia
  3. Establecer enlaces
  4. Crear tablas
  5. Utilizar formularios
  6. Aplicar estilo básico
  7. Agregar metadatos
  8. Crear estructura de navegación
